Statvoo is Advanced Realtime Website Analytics

== Description ==

This plugin adds the required javascript for Statvoo.com (Advanced Realtime Website Analytics).

For more information visit:

[Statvoo.com](http://www.statvoo.com/)

== Installation ==

1. Upload `statvoo` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. View (Admin > Settings > Statvoo.com) for further setup information
