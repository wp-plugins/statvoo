Enables Statvoo.com - Realtime Website Analytics and Traffic Monitoring - on all pages.

== Description ==

This plugin adds the required javascript for statvoo.com realtime website traffic monitoring.

For more information visit:

[Statvoo.com](http://www.statvoo.com/)

== Installation ==

1. Upload `statvoo` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. View (Admin > Settings > Statvoo.com) for further setup information
